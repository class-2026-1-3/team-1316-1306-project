# team-1316-1305-project===
백룸위키

우리가 자주 사용하는 나무위키를 참고하여 얼마 전에 영화로도 나왔던 인기있는 크리피파스타(도시전설)을 위키로 만들었습니다.
이미 위키로 존재하지만, 우리에게 친근한 나무위키를 참고하여 매우 익숙한 맛이기에 사용자가 더 편하게 이용할 수 있을 것입니다.
대문 : http://127.0.0.1:5500/html/wiki.html
----------------------------
구현된 창
1. 위키 대문
2. 예시 문서 (레벨 0)
3. 최근 변경
4. 로그인 / 회원가입 창
5. IP에러 창 (잘못된 접근)

구현된 기능
1. 위키 아이콘을 누르면 대문으로 이동됨
2. 최근 변경을 누르면 최근 변경 창으로 이동됨
3. 검색에 레벨 0을 검색하면 이동됨
4. 드롭다운 기능 추가
5. 사용자 아이콘을 누르면 로그인 / 회원가입 창으로 이동됨
6. 로그인 / 회원가입 창에서 IP에러 창 구경하러 이동 가능
